Модуль №4: Постановка задачи

Цель модуля

Изучить коллективные операции в MPI и способы создания новых коммуникаторов. Решить задачи, включающие использование MPI_Scatter, MPI_Gather, MPI_Bcast и MPI_Comm_split.

Формулировка задачи

Разработать программу, которая:

  1. Разделяет массив данных между процессами с помощью MPI_Scatter.
  2. Обрабатывает данные на каждом процессе.
  3. Собирает обработанные данные обратно с использованием MPI_Gather.
  4. Создает подгруппы процессов на основе рангов с помощью MPI_Comm_split.
  5. Выводит результаты на главный процесс и отображает информацию о новых группах.

MPIBegin73

В каждом процессе, ранг которого делится на 3 (включая главный процесс), даны три целых числа. С помощью функции MPI_Comm_split создать новый коммуникатор, включающий процессы, ранг которых делится на 3. Используя одну коллективную операцию пересылки данных для созданного коммуникатора, переслать исходные числа в главный процесс и вывести эти числа в порядке возрастания рангов переславших их процессов (включая числа, полученные из главного процесса).
